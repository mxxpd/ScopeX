# ScopeX

ScopeX is a Nuxt app that helps estimate design project cost, generate a breakdown of services, and export a client-ready commercial proposal as PDF.

## Setup

Install dependencies:

```bash
npm install
```

## Development

Start the dev server:

```bash
npm run dev
```

## Checks

Run linting:

```bash
npm run lint
```

Run typecheck:

```bash
npm run typecheck
```

Run tests:

```bash
npm run test:run
```

## Production

Build the app:

```bash
npm run build
```
